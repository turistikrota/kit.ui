# Top Header

Top header is used to some contract information and login and register buttons.

Features:

- Left, right and fill slots

<Canvas>
  <TopHeader className='hidden md:flex'>
    <TopHeader.Left>
      <span className='dark:text-white'>some promotion text</span>
    </TopHeader.Left>
    <TopHeader.Right>
      <span className='dark:text-white'>your some link</span>
    </TopHeader.Right>
  </TopHeader>
</Canvas>

## Usage

```js
import TopHeader from '@turistikrota/ui/headers/top'

export default function MyDefaultTopHeader() {
  return (
    <TopHeader className='hidden md:flex'>
      <TopHeader.Left>some promotion text</TopHeader.Left>
      <TopHeader.Right>
        <Link
          href={getStaticRoute(locale).aboutUs}
          className='hover:opacity-80 transition-all duration-200 ease-in-out'
        >
          link content
        </Link>
      </TopHeader.Right>
    </TopHeader>
  )
}
```
