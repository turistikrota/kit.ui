# DesktopInfoBox

```jsx
import { DesktopInfoBox } from '@turistikrota/ui/accessibility/info'
```

Desktop info box component for displaying information about the desktop. It's actually uses tooltip component to display the information.

Please note that this component is not meant to be used directly, but rather as a part of the desktop component.
