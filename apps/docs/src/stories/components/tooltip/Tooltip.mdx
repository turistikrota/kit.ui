# Tooltip

```jsx
import Tooltip, { TooltipProvider } from '@turistikrota/ui/tooltip'
```

Tooltip component is used to display additional information when hovering over a target element.

<Alert type='warning'>The Position value is only used on the first render. It cannot be changed later.</Alert>

<Controls />

```jsx
import Tooltip, { TooltipProvider } from '@turistikrota/ui/tooltip'

export default function TooltipExample() {
  return (
    <TooltipProvider>
      <Tooltip content={<span className='dark:text-gray-200'>this is tooltip content</span>}>
        <span className='dark:text-gray-300'>Hover Me</span>
      </Tooltip>
    </TooltipProvider>
  )
}
```
