# useToast()

A hook to show a toast notification.

## Usage

Before using the `useToast` hook, you need to add the `ToastProvider` to your application.

```jsx
import { ToastProvider, ToastListProvider } from '@turistikrota/ui/toast'

const App = () => (
  <ToastListProvider>
    <ToastProvider>
      <YourApp />
    </ToastProvider>
  </ToastListProvider>
)
```

Then you can use the `useToast` hook to show a toast notification.

```jsx
import { useToast } from '@turistikrota/ui/toast'

const YourComponent = () => {
  const toast = useToast()

  return <button onClick={() => toast('Hello world!')}>Show toast</button>
}
```

      Error

    <Button variant='primary' onClick={() => toast.info('Info')}>

      Warning
    </Button>
  </div>
  <div className='flex gap-5'>
    <Button
      variant='success'
      onClick={() =>
        toast.askSuccess({
          title: 'Are you sure?',
          cancelText: 'No',
          confirmText: 'Yes',
          description: 'This action cannot be undone.',
          onCancel: () => toast.info('Cancelled'),
          onConfirm: () => toast.success('Confirmed'),
        })
      }
    >
      Ask Success
    </Button>
    <Button

    >
      Ask Error
