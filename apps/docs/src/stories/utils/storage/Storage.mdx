# Storage

Storage util is have a functions for work with localStorage and sessionStorage.

## safeStorageParse

This function is safe parse JSON string from storage. If JSON string is not valid, not throw error and not call callback function.

```js
import { safeStorageParse } from '@turistikrota/ui/utils/storage'

safeStorageParse(localStorage.getItem('user'), (user) => {
  console.log(user)
})
```
