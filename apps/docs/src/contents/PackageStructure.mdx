# Package Structure

Turistikrota cares about the performance of your site and does not import any components unless you use it.

Importing components in Turistikrota is as follows:

```jsx
import Button from '@turistikrota/ui/button'
import Input from '@turistikrota/ui/form/input'
```

You may want to use the following instead. We thought about you and this situation, we decided that; you shouldn't want.

```jsx
import { Button, Input } from '@turistikrota/ui'
```

## Importing Components

```jsx
import Button from '@turistikrota/ui/button'
import Input from '@turistikrota/ui/form/input'
```

## Importing Hooks

```jsx
import { useSSR, useClient } from '@turistikrota/ui/hooks/ssr'
import { useListener } from '@turistikrota/ui/hooks/dom'
```

## Importing Utils

```jsx
import { wait } from '@turistikrota/ui/utils/async'
import debounce from '@turistikrota/ui/utils/debounce'
import { parseApiError } from '@turistikrota/ui/utils/response'
```
