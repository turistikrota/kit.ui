# Colors

`@turistikrota/ui` uses tailwind css. In addition to all tailwindcss colors the following can be used.

## Background Colors

Background colors are used for the background of elements, components, and screens.

use with `bg-` prefix.

Like: `bg-default`, `bg-secondary`, `bg-third`

<ColorPalette>
  <ColorItem
    title='light.theme.background-color.default'
    subtitle='Default BG'
    colors={{ default: 'rgb(239, 244, 245)' }}
  />
  <ColorItem title='light.theme.background-color.secondary' subtitle='Secondary BG' colors={{ default: '#fff' }} />
  <ColorItem title='light.theme.background-color.third' subtitle='Third BG' colors={{ default: '#e7edef' }} />

  <ColorItem
    title='dark.theme.background-color.default'
    subtitle='Default BG'
    colors={{ default: '#0f0e0e' }}
  />
  <ColorItem title='dark.theme.background-color.secondary' subtitle='Secondary BG' colors={{ default: '#121212' }} />
  <ColorItem title='dark.theme.background-color.third' subtitle='Third BG' colors={{ default: '#191919' }} />
</ColorPalette>

## Border Colors

Border colors are used for the border of elements, components, and screens.

use with `border-` prefix.

Like: `border-default`

<ColorPalette>
  <ColorItem
    title='light.theme.border-color.default'
    subtitle='Default BG'
    colors={{ default: '#f2f0f0' }}
  />

  <ColorItem
    title='dark.theme.border-color.default'
    subtitle='Default BG'
    colors={{ default: '#252525' }}
  />
  
</ColorPalette>
