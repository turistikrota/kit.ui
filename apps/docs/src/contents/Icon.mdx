Turistikrota uses [boxicons](https://boxicons.com/) as its icon library.

## Example usage

add boxicons to your project

```bash
yarn add boxicons
```

and import boxicons's css file in your main package:

```js
import 'boxicons/css/boxicons.min.css'
```

then you can use boxicons in your project:

```jsx
function App() {
  return (
    <div>
      <i className='bx bx-home'></i>
    </div>
  )
}
```
